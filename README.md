# Kasa

Application web de location immobilière développée avec React et React Router, avec une interface moderne et responsive.

## Description du projet

Kasa est une plateforme de location immobilière qui permet aux utilisateurs de :

-   Consulter une liste de logements disponibles
-   Visualiser les détails de chaque logement (photos, titre, localisation, équipements)
-   Accéder à des informations sur l'entreprise via la page "À propos"
-   Naviguer facilement entre les différentes sections

Le projet a été développé en respectant des maquettes Figma responsives et des guidelines techniques précises, avec une architecture moderne et des composants réutilisables.

## Comment démarrer le projet

### Prérequis

-   Node.js (version 16 ou supérieure)
-   npm ou yarn

### Installation

1. Clonez le dépôt :

```bash
git clone https://github.com/MaximeC31/Kasa-react.git
cd Kasa-react
```

2. Installez les dépendances :

```bash
npm install
```

3. Lancez le serveur de développement :

```bash
npm run dev
```

Le projet sera accessible à l'adresse : `http://localhost:5173`
